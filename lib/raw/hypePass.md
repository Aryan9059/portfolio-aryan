# **hype.pass**

hype.pass allows you to store your passwords and card information locally on your device's storage without requiring any internet permissions. Your data remains securely encrypted and can only be accessed by you using biometrics or an app specific pin.

## 🚀 Features

- 🗝️ **Save Passwords securely offline on your phone**
- 💳 **Save your Card details securely**
- 🫰 **Use Biometric Authentication**
- ✨ **Auto generate strong Passwords**
- ✳️ **Encrypted using AES-265 & Android Keystore**
- ⏬ **Create encrypted backups & Restore them using your PIN**
- 🚫 **Screenshots blocked from the app**
- #️⃣ **Recover forgotten PIN using Recovery Key**
- 🎬 **Smooth Animations**
- 🎨 **Material 3 based Modern UI**
- 🌚 **Available in Light/Dark Mode**

## 🧑🏻‍💻 Tech Stack & Tools Used

- Android Kotlin
- Jetpack Compose
- MVVM Architecture
- Room Database
- Kotlin Coroutines
- Jetpack Navigation

## Getting Started

To get started with the hype.pass App, simply download the app from the Releases page of the Repository or download the source code and build it yourself.

### Prerequisites

- Android Studio
- Git
- Room Database
- Jetpack Navigation

### Installing

1. Clone the repository
``` git clone https://github.com/Aryan9059/hype.pass.git ```
2. Open the project in Android Studio
3. Build and run the app
