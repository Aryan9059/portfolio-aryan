# **Phi Launcher**

Phi Launcher (formerly Hype Launcher) is a lightweight, minimalist Android launcher built for speed, customization, and productivity. Keep your home screen clean, launch apps instantly, and enjoy a distraction-free experience with zero bloat.

## 🚀 Features

- **Phi Widgets:** Quick Notes, To-Do list, and Music Controller accessible with a swipe.
- **Custom Home Screen:** 5 clock styles, search bar options, shortcuts, folders, and pinned apps.
- **Dev Panel:** Track LeetCode, Codeforces stats, and friends' coding progress instantly.
- **Smart App Drawer:** List/grid view, alphabetical scroller, usage stats, folders, and quick actions.
- **Deep Customization:** Drawer opacity, grid columns, icon packs, alignment, shuffle mode, and more.
- **Focus & Privacy:** Hide apps, minimal mode, focus mode, launch delay, and screen time insights.
- **Fast Performance:** Zero ads or trackers, smooth animations, instant launches, lightweight size.
- **Accessibility Feature:** Uses AccessibilityService only for **Double Tap to Sleep**, with no data collection.

## 🧑🏻‍💻 Tech Stack & Tools Used

- Android Kotlin
- Jetpack Compose
- MVVM Architecture
- Room Database
- Kotlin Coroutines
- Compose Navigation
- Retrofit
- GraphQL Queries
- DataStore

## Getting Started

To get started with Phi Launcher, download the app from the Releases page or build from source.

### Prerequisites

- Android Studio
- Git

### Installing

1. Clone the repository

```bash
git clone https://github.com/Aryan9059/HypeLauncher.git
```

2. Open the project in Android Studio
3. Build and run the app
