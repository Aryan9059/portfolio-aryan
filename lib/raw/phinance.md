# PhiNance

**PhiNance** is a modern Android finance tracker built with **Kotlin** and **Jetpack Compose**, designed to make expense tracking fast, simple, and intuitive.

Instead of filling out lengthy forms, users can log expenses using natural, command-style input:

```text
120 food
250 groceries
```

PhiNance is currently under active development and will be published on the Google Play Store under the **Phi Apps** account.

---

## Why PhiNance?

Most expense trackers require multiple taps and screens to record a single transaction. PhiNance focuses on reducing friction by making expense logging as quick as typing a short message.

With PhiNance, users can:

* Add expenses in seconds
* Track spending across categories
* Monitor financial habits
* Manage shared expenses with friends and groups
* Access additional finance tools through future updates

## Features

### Fast Expense Entry

Log expenses instantly using simple text commands.

```text
120 food
```

No complex forms. No unnecessary steps.

### Category-Based Tracking

Organize transactions into categories to better understand spending patterns.

### Shared Expenses

Manage group expenses with a Splitwise-inspired system for friends, roommates, and teams.

### Modern Android Experience

Built entirely using modern Android development practices:

* Kotlin

## Architecture

PhiNance follows a clean MVVM-based architecture.

### Presentation Layer

* Compose screens
* Reusable UI components
* ViewModels
* Navigation

### Domain Layer

* Business logic
* Use cases
* Core models

### Data Layer

* Repositories
* Local storage
* Database management
* Preferences handling

This separation ensures scalability, maintainability, and easier feature expansion.
